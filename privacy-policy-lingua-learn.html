Lingua Learn Privacy Policy

Last updated: April 20, 2026

This policy applies to Lingua Learn, provided by Breanna Hansen DBA Cima Development.

Lingua Learn is currently in production. This policy may be updated as features evolve.

Planned Data Categories

  • Learning progress data (for example, flash card results, memory game outcomes, and vocabulary practice).
  • Level and preference settings.
  • If speaking features launch, voice interaction data needed to provide and improve that feature.
  • Technical diagnostics and performance data.

Planned Data Use

  • Provide personalized Spanish learning experiences.
  • Adapt learning flows to user level and progress.
  • Improve speaking and practice features over time.
